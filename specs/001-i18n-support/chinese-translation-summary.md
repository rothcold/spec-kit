## 使用方法

### 自动语言选择

当用户使用 `--lang zh_CN` 时，系统将自动使用这些中文命令模板:

```bash
# 使用中文
specify --lang zh_CN

# 所有命令将显示中文帮助和输出
```

### 模板文件路径

- **英文原版**: `templates/commands/*.md`
- **中文翻译**: `templates/i18n/zh_CN/commands/*.md`

### 系统行为

`get_template_path()` 函数将根据语言设置自动选择:

1. 如果 `--lang zh_CN` → 使用 `templates/i18n/zh_CN/commands/`
2. 如果文件不存在 → 回退到 `templates/commands/` (英文)
